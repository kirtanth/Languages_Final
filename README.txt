---------------------Languages Web Application-------------------------
This is Simple Application which gives you Information about Languages You Select.
it Gives you the Latest Version of the language and Topics of That particular Language

-----------------------How to Start Application-------------------------

Install Node from : https://nodejs.org/

-open IDE(Visual studio Code Recommended)
- open Terminal - npm i
		- npm start
- open browser : http://localhost:3000/  ||  http://127.0.0.1:3000/
